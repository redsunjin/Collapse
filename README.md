## 🎭 프로젝트 소개 (Introduction)

**붕괴(Collapse)**는 양자역학의 원리를 서사 구조로 치환한 인터랙티브 소설입니다. 독자인 당신이 텍스트를 읽고 선택을 내리는 순간, 확률의 파동으로 존재하던 수많은 평행 우주 중 단 하나의 현실이 **'붕괴(Collapse)'**되어 확정됩니다.

이곳은 단순한 저장소가 아니라, 우주의 운명을 결정하는 **양자 관측 터미널**입니다.

## 📡 시스템 특징 (Key Features)

*   **Quantum State HUD**: 현재 타임라인의 응집도(Coherence)와 중첩 상태를 실시간으로 모니터링합니다.
*   **Probabilistic Branching**: 당신의 의지가 개입하기 전까지 이야기는 정해진 길을 가지 않습니다.
*   **Glassmorphism UI**: 미래형 관측 장비를 모티브로 한 세련되고 몰입감 넘치는 UI를 제공합니다.

---

## 📖 관측 가이드 (Observer's Manual)

1.  **관측(Observe)**: 화면에 나타나는 텍스트의 중첩 상태를 신중히 읽으십시오.
2.  **확정(Collapse)**: 하단의 액션 버튼을 클릭하여 파동함수를 붕괴시키고 현실을 확정하십시오.
3.  **재구성(Recollapse)**: 타임라인이 불안정해지면 우측 상단의 리셋 기능을 사용해 새로운 가능성을 탐색하십시오.
